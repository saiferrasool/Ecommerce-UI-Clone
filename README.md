# Ecommerce UI Project (Amazon Inspired Clone)

## Description
This is a **responsive e-commerce front-end project** inspired by Amazon's UI.  
It is built purely using **HTML and CSS**, including **flexbox layouts, media queries, and hover effects** to replicate a modern e-commerce site experience.  

> ⚠️ This project is for **learning and practice purposes only**.  
> It is **not affiliated with Amazon** and does not contain any real products or branding.

## Features
- Responsive navigation bar with location, search, sign-in, and cart sections
- Multi-card product layout with images and links
- Hover effects on cards, images, buttons, and navigation
- Hero banner with overlay text and link
- Responsive design for **desktop, tablet, and mobile**
- Clean and modern UI using **Google Fonts** and **Font Awesome icons**

## Tech Stack
- **HTML5**
- **CSS3**
- **Flexbox & Media Queries**
- **Google Fonts (Inter, Roboto)**
- **Font Awesome Icons**
